I Built a web based School Management System using Java. The app manages students and admins. It follows MVC structure and runs on Apache Tomcat.

Tech Stack
• Java
• JSP
• Servlet
• JDBC
• PostgreSQL
• Maven
• Apache Tomcat

Features
• Admin login
• Add student details
• View student list
• Update student data
• Delete student records

Project Structure
• src main java – controllers, services, entities
• src main webapp – JSP files
• WEB-INF – web.xml
• pom.xml – Maven dependencies

How to Run Locally
• Install JDK 8 or higher
• Install Apache Tomcat 9
• Install PostgreSQL
• Create database
• Update DB credentials in code
• Import project in Eclipse
• Run on Tomcat server
• Open browser
• http://localhost:8080/SchoolManagementSystem

Database
• PostgreSQL
• Tables: admin, student

Future Improvements
• Password hashing
• Role based access
